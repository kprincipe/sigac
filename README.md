# Sistema de Gestão Acadêmica
Este projeto foi desenvolvido como exercício para os tópicos de `alocação de memória dinâmica` e `manipulação de arquivos` da disciplina Estrutura de Dados.<br>

## Como funciona
O sistema conta com uma interface simples de texto para terminal, de modo que se possa cadastrar, remover ou editar informações de discentes, cursos e turmas (assim como discentes matriculados nas turmas). Todos os dados são registrados e lidos com arquivos texto .csv (comma separated values) não encriptados. 

## Como compilar?
- Não exige nenhuma dependência além do compilador de linguagem C (gcc-14.2.0) e a biblioteca padrão glibc.
```bash
$ make
$ ./main
```
