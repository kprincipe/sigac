# Sistema de Gestão Acadêmica
Este projeto foi desenvolvido como exercício para os tópicos de `alocação de memória dinâmica` e `manipulação de arquivos` da disciplina Estrutura de Dados.

## Como compilar?
- Não exige nenhuma dependência além do compilador de linguagem C (gcc-14.2.0) e a biblioteca padrão glibc.
```bash
$ make
$ ./main
```
